# Hospital Management System (CRUD) with Flask

This project is a **Hospital Management System** built using **Python** and **Flask**. 
It demonstrates **CRUD (Create, Read, Update, Delete) operations** for managing hospital data including patients, doctors, and appointments. 
The system provides a **user-friendly web interface** to perform all operations efficiently.

## Features
- Add, view, update, and delete patient records.
- Manage doctor information and appointments.
- Dynamic web pages using Flask templates (HTML/CSS).
- Database integration for persistent data storage.
- Easy to extend for additional modules like billing or pharmacy management.

## Technologies Used
- Python 3.x
- Flask (Web framework)
- SQLite / MySQL (Database)
- HTML / CSS (Frontend)
- Jinja2 templates
